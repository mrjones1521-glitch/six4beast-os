      Build Steps on HP → iPhone

      1. Download and unzip this folder on your HP laptop.
      2. Install VS Code.
      3. Open this folder in VS Code.
      4. Install the VS Code extension called Live Server.
      5. Right-click index.html and choose Open with Live Server.
      6. Test the app on your HP browser.
      7. To use on iPhone, upload the folder to GitHub Pages, Netlify, or Vercel.
      8. Open the live link in iPhone Safari.
      9. Tap Share → Add to Home Screen.
      10. Use Apple Watch Workout app separately for heart rate until native Watch build later.
